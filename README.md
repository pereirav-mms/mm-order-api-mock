# mm-order-api-mock\n\nEstructura de archivos para GitHub Pages mock API.\n\nCada pedido se sirve como un archivo estático en: \n\nhttps://pereirav-mms.github.io/mm-order-api-mock/orders/number/{order}/email/{email}/index.json\n\nEjemplos:\n- orders/number/1201/email/dummy.customer1@test.com/index.json\n- orders/number/1208/email/dummy.customer2@test.com/index.json\n\nGitHub Pages devolverá 404 automáticamente si la URL no existe, por lo que no es necesario un endpoint especial para pedido no encontrado.\n
